# Project Overview

This is a minimal Astro project, based on the "minimal" starter kit. It appears to be a web application, likely a personal website or portfolio, given the "mytopten" directory name.

The project uses Astro, a modern static site builder. The file structure follows the standard Astro conventions:

*   `src/pages`: Contains the pages of the site.
*   `src/components`: Contains reusable components.
*   `src/layout`: Contains layout components.
*   `public`: Contains static assets like images, CSS, and fonts.

# Building and Running

The following commands are available to run the project:

*   `npm install`: Installs dependencies.
*   `npm run dev`: Starts the local development server at `localhost:4321`.
*   `npm run build`: Builds the production site to the `./dist/` directory.
*   `npm run preview`: Previews the built site locally before deploying.

# Development Conventions

*   The project uses TypeScript, as indicated by the `tsconfig.json` file.
*   The project uses the standard Astro file-based routing.
*   Components are written as `.astro` files.
